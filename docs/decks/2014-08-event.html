2014夏 AL作戦／MI作戦

「いざミッドウェー！　一航戦・二航戦を活躍させるぞ！」と意気込んで臨んだが、お札システムにより出撃タイミングを見極められず二航戦はずっとお留守番。最終ステージはミッドウェーでなく本土奇襲。そのため、一航戦・二航戦に出番はなかった。イベント開始前のテンションから考えると不完全燃焼。一航戦・二航戦が主役の第二次ミッドウェーイベントを期待したい。
　イベント難易度は高め。E-1から難しかった記憶。敵編成がランダムで潜水艦／水上艦に切り替わるマスがあったり、イベント開始時に伏せられてた札があったり、厄介なイベント。

過去の記録編。2014年夏・E-6クリア。
初の連合艦隊で挑むミッドウェー決戦。
札システム初登場。今より厳しい仕様で、支援艦隊は札なし艦限定。「AL」「MI」札の後に隠し札「本土」があり、ミッドウェーに全力投入したRTA勢を薙ぎ払った。
